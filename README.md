

# Secure Registration Automation (CTF)

## Overview

This project demonstrates a Python-based automation solution for a secured registration portal protected by advanced anti-bot and anti-replay mechanisms.

The challenge required replicating real browser behavior, including JavaScript execution and session cookie generation, which cannot be achieved using traditional HTTP libraries alone.


## Installation Setup

* Open Folder in Code Editer (Vs Code)

* Create Virtual Enviroment and activate it

  ` python -m venv venv
  `

  ` vanv/Scripts/activate
  `

* Download requirements

  `python -m pip install greenlet==3.3.0 playwright==1.57.0 pyee==13.0.0 typing_extensions==4.15.0
   `

* Run the Script

  `python solve.py
  `
